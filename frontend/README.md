# AI Analyst Pro - Frontend

React single-page application for AI Data Insights Analyst.

## Setup

1. **Install dependencies:**
   ```bash
   cd frontend
   npm install
   ```

2. **Start development server:**
   ```bash
   npm start
   ```

The app will run on `http://localhost:3000`

## Features

- Single-page React application
- Drag & drop CSV file upload
- Real-time processing status
- Download and view generated reports
- Dark theme with mature color palette

## Backend API

Make sure the Flask backend is running on `http://localhost:5000`
